Description

  This task focuses on styling a personal profile web page using CSS with an emphasis on understanding and applying the CSS box model. The styling enhances the visual structure and readability of the page while following best practices in CSS development.

Work Performed

  Created a separate external CSS file (style.css) and linked it to the HTML document.

  Applied global styles to define consistent font family, background color, and text color across the page.

  Implemented the CSS box model by using margin, padding, and border properties to control spacing and layout.

  Styled headings, paragraphs, lists, and navigation elements to improve visual hierarchy and readability.

  Used class-based styling instead of inline styles to maintain clean, scalable, and reusable CSS code.

  Inspected and adjusted layout spacing using browser developer tools to understand box model behavior.

Technologies Used

  HTML5

  CSS3

  Outcome

The profile page is visually structured with clear spacing and layout, demonstrating a proper understanding of the CSS box model and its role in web page design.
